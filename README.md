## 🚀 Overview

**Evkira** is a modern house rental platform UI that focuses on simplicity, usability, and clean design.
It allows users to explore properties, view details, and interact with listings in a smooth and intuitive interface.

This project is built as a **frontend-first prototype**, with future plans for full backend integration.

---

## ✨ Key Features

* 🔍 Smart property search & filtering
* 🏘️ Clean and modern interface
* 📱 Fully responsive design (mobile-friendly)
* ❤️ Favorites system (UI)
* 📅 Appointment booking system (UI)
* 🌙 Dark mode support
* ⚡ Smooth animations and transitions
